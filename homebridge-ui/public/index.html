AiSEG2 Controller

Fixed AiSEG2 IP or hostname. Leave empty to use local discovery.

Apple Home Layout

These options keep related sensors grouped under the same accessory tile in Apple Home.

Direct ECHONET Lite Control

Uses AiSEG2 for names and discovery, then talks directly to matched ECHONET Lite devices for better capabilities when available.
Optional comma-separated CIDR targets. Empty scans local private subnets.

ECHONET Diagnostics

For troubleshooting only. It does not change which devices are controlled directly.

Energy Status in Apple Home

Shows Detected/Not Detected status sensors from ECHONET solar and storage battery data, plus battery level when available. Raw W/kWh values are logged only.

Homepage Status API

Read-only JSON endpoint for Homepage customapi widgets. It does not add services to Apple Home.

EcoCute Solar Start

Starts manual water heating when solar, battery, weather, and time window rules pass. The plugin does not send stop.
Use 0 to allow standby or low charging.
Requires HTTPS or localhost. Otherwise enter coordinates manually.

Door Lock Webhook

Local HTTP endpoint for external systems such as UniFi fingerprint events.
